PICAutoDroid

Android-Controlled Mobile Robot

An innovative mobile robotics project featuring wireless Android control, and voice-controlled autonomous navigation capabilities using PIC18F452 microcontroller technology.

Hardware Architecture

Advanced robotics platform built with professional-grade components

PIC18F452 Microcontroller

Main processing unit handling sensor data, motor control, and communication protocols.

40MHz 32KB Flash 1.5KB RAM

L298N Motor Driver

Dual H-bridge motor controller for precise speed and direction control of DC motors.

2A Per Channel PWM Control Bidirectional

HC-05 Bluetooth Module

Wireless communication interface enabling Android device connectivity and control.

Class 2 10m Range UART

Wireless Camera

Real-time video streaming capability for remote monitoring and navigation assistance.
