PICAutoDroid, Mobile Robot using PIC18F452 8-bit microcontroller

December 1, 2012 to March 15, 2013
4 minutes read

PICAutoDroid was my hobby project. The brain of the robot is the PIC18F452 8-bit microcontroller coupled with RN-42 Bluetooth module. The robot is controlled by an android app. The app reads the built-in gyroscope. The control commands to the robot are the function of gyroscope sensor data. The orientation of the robot depends on the device orientation. The commands are sent to the robot over wireless Bluetooth communication link. Also, the robot is commanded through voice commands.
